## 1. 商業模式：訂閱制 vs 買斷制

### 業界現況

**結論：美業預約系統 99% 採訂閱制，買斷制幾乎絕跡。**

| 模式 | 優點 | 缺點 |
|------|------|------|
| **訂閱制（推薦）** | 入門門檻低、持續更新、彈性擴展、雲端服務、即時客服、資安持續維護 | 長期累積成本對客戶較高 |
| **買斷制（不推薦）** | 一次付清、長期成本可預測 | 初始投入高、版本固定、無支援、資安風險、無法擴展 |

### 為什麼訂閱制是正解

1. **店家現金流壓力小** — 1008 老闆明確表示：「好幾萬塊太貴了，比較適合連鎖」
2. **LINE API 會變動** — Messaging API、LIFF SDK 持續更新，需要維護
3. **金流選項會增加** — LINE Pay、街口、信用卡等需要陸續整合
4. **規模擴張順勢升級** — 從 1 人店到 3 人店，從單店到連鎖，訂閱方案可彈性升級
5. **SaaS 財務模型健康** — 持續收入 → 持續投入 → 持續改善 → 持續收入（正循環）

- [破除軟體部署迷思](https://blog.jandi.com/tw/subscription-v-s-buyout/)

## 2. 國際競品分析

**定位：** 全球最大免費沙龍預約平台
**定價模式：** Pay-as-you-go（免費 + 交易手續費）
- 軟體免費，無月費

## 4. 定價策略建議

### 4.1 市場價格區間總覽

```
NT$0 ─────── NT$399 ────── NT$1,000 ─── NT$3,000 ─── NT$5,000 ── NT$10,000+
夯客免費    樂創約進階     夯客中階      夯客商務      1008 精品    Mangomint
Fresha       Booksy 單人    ?            ?            ?            國際高端
```

**台灣市場觀察：**
- **NT$399 以下：** 超小店、免費試用
- **NT$400-3,000：** 一般小店（夯客、樂創約的主戰場）
- **NT$3,000-10,000：** 精品小店、中店（1008 目標區）
- **NT$10,000+：** 中大型店、連鎖（17FIT、美業歐巴進階）

### 4.2 1008 Hair Studio 定價策略

基於老闆訪談（「NT$3,000-10,000 可接受，1 萬以下都行」）+ 競品研究：

| 層級 | 定價 | 對象 | 功能 |
|------|------|------|------|
| **免費試用** | 0/月（30 天） | 新客戶 | 全功能試用，限 30 天 |
| **標準方案** | **NT$2,000/月** | 1 人小店 | 核心預約 + LINE 整合 + 基本 CRM |
| **精品方案** | **NT$5,000/月** | 1-3 人店（1008 目標）| 全功能 + 客製化設定 + 專屬支援 |
| **企業方案** | 洽詢 | 4+ 人店 / 多店 | 多設計師、進階分析、API |

### 4.3 為什麼選這個定價

- **NT$2,000 標準方案：** 比夯客 NT$300 貴 6 倍，但提供明顯更好的體驗和支援。夯客的續約率 99% 代表店家願意留在系統，但 NT$2,000 是精品市場進入門檻。
- **NT$5,000 精品方案：** 對標 Mangomint NT$5,300，但本地化。老闆訪談「1 萬以下都行」的甜蜜點。
- **差異化優勢：**
  1. LINE 深度整合（對標夯客）
  2. 客製化體驗（超越所有 SaaS 競品）
  3. 精品級設計（對標 GlossGenius）
  4. 手機優先 admin（對標 Booksy）
  5. 在地化支援（對標樂創約）

## 5. 預約窗口最佳實踐

### 5.1 業界建議

| 項目 | 業界範圍 | 理由 |
|------|---------|------|
| 最多提前 | 30-60 天 | 太遠會填滿「投機預約」導致高取消率 |
| 最少提前 | 24-48 小時（診所）/ 1-3 小時（理髮） | 給店家準備時間，但不能太嚴格 |
| 新客 vs 熟客 | 可設不同規則 | 新客需要更多準備時間 |

### 5.2 關鍵洞察

**不應硬編碼，應讓老闆在後台自己設定。**

理由：
1. **跨行業需求不同** — 診所 vs 理髮 vs 塔羅的預約習慣完全不同
2. **店家經驗不同** — 有的老闆喜歡長期預約，有的只接當週
3. **季節性** — 過年、節慶前要縮短預約窗口，避免投機預約

### 5.3 1008 Hair Studio 建議（預設值）

- **最多提前：30 天**（理髮不需要兩個月前預約）
- **最少提前：1 小時**（給 Redis lock + 老闆準備餘裕）
- **後台可設定：是**（老闆可依季節調整）

## 6. Onboarding Tour 最佳實踐

### 6.1 核心原則

**重要前提：** 1008 是完全客製化的模式。系統設定（店名、地址、服務項目、營業時間、LINE 帳號）在訪談時由開發者完成，老闆不需要自己設定。因此 Onboarding 不是「幫老闆設定系統」，而是**「帶老闆體驗系統怎麼幫他」**。

目標：**降低 Time to Value** — 讓老闆在最短時間內感受到「這個東西真的能幫我」。

1. **體驗導向** — 帶老闆看他的真實資料（今天的預約、他的服務列表），不是看空白設定頁
2. **痛點對應** — 每個步驟都連結到他說過的痛點（「花太多時間回訊息」「手機和 Excel 來回切換」）
3. **動作導向** — 讓老闆在真實介面上點擊操作，不是看說明
4. **進度條** — 讓老闆看到還剩幾步
5. **永遠可 Skip** — 尊重老手

### 6.2 1008 Hair Studio 體驗導覽流程

**場景：** 系統已預設完成（開發者在訪談時建好），老闆第一次打開後台。

```
Step 1/5: 歡迎！這是你今天的行事曆
        → 老闆看到：今天有 3 個預約，時間、客人名、服務一目了然
        → 對應痛點：「不用再開 Excel 對照了」
        [進度條 ▓▓░░░░░ 20%]

Step 2/5: 試試看！點一下這個預約
        → 老闆點擊 → 彈出客人詳情：姓名、過去來幾次、上次做什麼、備註
        → 對應痛點：「不用再問客人上次做什麼」
        [進度條 ▓▓▓░░░░ 40%]

Step 3/5: 客人剪完了？滑一下標記「完成」
        → 老闆操作 → 預約狀態更新
        → 對應痛點：「不用再手動記錄誰來了誰沒來」
        [進度條 ▓▓▓▓░░░ 60%]

Step 4/5: 看看你的客人在 LINE 裡看到什麼
        → 展示客人端預約畫面的截圖/模擬
        → 老闆理解：「哦，客人自己就會約了，不用再回訊息」
        [進度條 ▓▓▓▓▓░░ 80%]

Step 5/5: 完成！現在起，客人會自己預約
        → 摘要：你省下的時間 = 不用回訊息 + 不用開 Excel + 不用手動提醒
        → 「有任何問題直接 LINE 我」
        [進度條 ▓▓▓▓▓▓▓ 100%]
```

**注意：** 設定類操作（修改服務、調整營業時間、設定預約窗口）不在 Onboarding 裡。這些放在後台設定頁，老闆需要改的時候自己去改，或跟開發者說。

### 6.3 未來 SaaS 版 Onboarding（自助開店時才需要）

當產品擴展為 SaaS 多店版時，才需要設定型 Onboarding（填店名 → 綁帳號 → 建服務 → 設時段）。現在的客製化模式不需要。

## 8. 1008 Hair Studio 差異化定位

### 8.1 SWOT 分析

**優勢（Strengths）：**
- 程式碼完全控制（vs SaaS 固定 UI）
- 可為每家店做客製化體驗
- LINE 深度整合（對標夯客）
- 現代技術棧（Next.js 16 + Prisma 7）
- 多租戶架構已準備好

**劣勢（Weaknesses）：**
- 只有 1 個客戶
- 品牌認知度低
- 無行銷預算
- 一人開發維運

**機會（Opportunities）：**
- 台灣美業市場仍在成長（夯客從 1,000 → 3,000+ 家）
- 中高端定位區有空白（NT$3,000-10,000/月）
- 設計美感差異化（GlossGenius 在台灣沒對手）
- 精品管理服務模式（一家一家服務）

**威脅（Threats）：**
- 夯客續約率 99%，切換成本高
- 新進者樂創約價格戰（NT$399/月）
- LINE API 變動風險
- 金流法規複雜度

### 8.2 定位金句

> **「不是更便宜的系統，是更美、更順、更懂你店的系統。」**

### 8.3 前三個差異化重點

1. **客製化體驗** — 每家店都是獨特的設計，不是統一模板
2. **LINE + Mobile-first Admin** — 客人端和老闆端都專為手機和 LINE 優化
3. **精品管理服務** — 不是自助 SaaS，是 premium onboarding + 專屬支援

### 8.4 避免的陷阱

- **不打價格戰** — 永遠輸給 NT$399/月
- **不做全功能** — 永遠輸給夯客的 3,000+ 店驗證
- **不做通用型** — 永遠輸給 Acuity
- **只做精品路線** — 在自己的規則內競爭
