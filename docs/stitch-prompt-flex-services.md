## 微調用 Follow-up 指令（視需要使用）

如果某張畫面的圖片不滿意，選取該畫面後單獨下指令：

```

把這張畫面的圖片改成：[描述你想要的氛圍]，保持其他設計不變

# 換描述文字
把描述改成「XXX」，其他不動

# 整體微調（選取所有畫面後使用）
所有畫面的按鈕圓角改成 4px，背景統一確認是 #FFF8F1 不是純白
```
