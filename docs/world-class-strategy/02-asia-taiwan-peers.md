# 亞洲美業預約系統／平台商業模式演進深度研究

> 研究範圍：日本、韓國、中國、台灣、東南亞
> 聚焦：商業模式演進、take rate、市場結構差異、對台灣 LINE 生態下 SaaS 的啟示
> 產出日期：2026-04-14

---

## 一、日本：Recruit 王國與「雜誌基因」

### 1.1 Hot Pepper Beauty（HPB）— 雜誌演化出來的平台霸主

**起點**：Recruit 旗下 Hot Pepper 原本是 2001 年前後從 Recruit 的免費情報誌分化出來、每月派發的**免費折價券雜誌**。2008 年前後將美容分類獨立為「Hot Pepper Beauty」，逐步遷移到 App。獨特的「**紙本折價券 → 網站 → App → 預約平台**」演進路徑。

**規模（2020）**：
- 店家數 **10 萬+**
- 年線上預約 **1 億件**
- 會員數 **2,464 萬人**
- 平均 **約 60% 新客來自 HPB**

**商業模式（雙層收費）**：
1. **主要營收 = 刊登費（廣告模式）**。7 檔刊登方案，月費從 Light 約 **25,000 日圓**，到都心 Platinum LL **30 萬日圓+**
2. **預約手續費 = 交易金額的 2%**（極低）
3. **Recruit Points 回饋 2%**，閉環回集團內

**為什麼強到變基建**：現金流主體是「廣告位置費」不是「GMV 抽成」。不論有沒有來客，店家都要付月費。這是雜誌廣告思維的延伸。**「廣告費是主體、交易費只是附帶」的 take rate 結構**，是其他市場難以複製的關鍵。

### 1.2 SalonBoard — HPB 的真正護城河

Recruit 自家美業**後台 SaaS**（預約、CRM、POS、排班）。2012 年推出，2022 滿 10 週年，全國 **5 萬家店在用**。

**核武等級設計**：對 HPB 刊登店家**完全免費**。店家進 HPB 拿客 → 強制用 SalonBoard → 資料全沉澱 Recruit 雲端 → **離開 HPB 等於放棄所有客戶 CRM** → 轉換成本近乎無限。

### 1.3 minimo（mixi 旗下）— 指名經濟的 marketplace

2014 年 mixi 推出，原本是「カットモデル募集 App」，後演化為「**直接指名個人設計師**」的 marketplace。

**商業模式（完全相反 — 向個人設計師收費，不向店家收費）**：
- 2,000 日圓以下收 300 日圓、2,001 日圓以上收 500 日圓
- take rate 約 **5-10%**
- 訂閱費、刊登費 0 元

**規模（2025）**：App 下載 **750 萬**、月預約 **40 萬**、美業工作者 **2.7 萬人**。

**戰略意義**：打「**設計師個人品牌」而非「店家品牌」**。

### 1.4 STORES 予約（前 Coubic）

2013 年 Coubic 起家做泛用預約 SaaS，2020 被 STORES（hey inc.）整合。純訂閱制，沒有 GMV 抽成。走「不跟 Recruit 正面打美業，靠泛用性抓長尾」路線。

### 1.5 Rakuten Beauty、EPARK

雙雙被 HPB 的網絡效應壓制。日本美業被 Recruit 壟斷到幾乎沒有第二名縫隙。

## 二、韓國：超級 App 主場，獨立 SaaS 幾乎沒空間

### 2.1 Naver 예약

Naver 本質是廣告搜尋公司，**預約只是把搜尋留存度做深的工具**。
- 刊登費、月費 **0 元**
- 交易手續費 **3.19%**（整合 Naver Pay 後合計約 3.74%）
- 真正 monetization 是搜尋廣告

### 2.2 Kakao Hairshop（현 마메드네）

2016 年以 KakaoTalk 11 億用戶推出。**與 Naver 相反極端**：
- **Take rate 約 15%**
- 入點費 5 萬韓元 + 月費 2 萬韓元
- **100% 預付制**（no-show 從 20% → 0.5%）
- 關鍵：「Style Book」— Instagram 式 discovery + 預約閉環

但 15% take rate + 預付制讓小店望而卻步，常「**請改用 Naver 別用 Kakao**」。

### 2.3 結構性啟示

Naver + Kakao 雙頭寡佔壓縮獨立 SaaS 空間。韓國沒有像 Recruit 這種中間層公司；流量 100% 交給超級 App，獨立玩家只做後台（공비서、핸드 SOS）。

## 三、中國：到店 vs 到家的歷史斷層

### 3.1 美團麗人

**規模（2024）**：
- 「大美業」（美業+醫美）**季度 GMV ~700 億 RMB，年化 ~2,800 億**
- 傳統美業抽成 **4-5%**，總貨幣化率 ~7-8%
- 輕醫美抽成 **6-8%**，醫美整體貨幣化率 15-20%

**商業模式核心**：團購券（預售）+ 到店核銷 + 低抽成。靠走量，與 HPB 月刊登費完全相反。

### 3.2 河狸家 — O2O 到家美業的集體殞落

2014 雕爺創辦，以美甲師上門服務切入。2015 C 輪估值 10 億美元。

**為什麼失敗**：
1. 非標服務無法規模化
2. **抽成過高（約 20%）逼走手藝人**
3. 低頻、單價低、反向補貼黑洞
4. **47.2% 用戶因「安全顧慮」不選上門**
5. C 輪後 2016-2024 整整 8 年沒新融資

**58 到家、e 家潔、GoLife** 同類命運。

### 3.3 博卡、有贊美業、美圖 SaaS

- **博卡（2004）**：美髮系大型連鎖，整體市佔 **50%+**，北上達 80%
- **有贊美業（2012）**：2021 年付費商家 5,000+
- **美圖 SaaS**：醫美 + AI 試妝

中國美業 SaaS 呈「博卡守連鎖、有贊守電商、美圖守醫美」三足，**沒有 Recruit 式單一霸主**。

### 3.4 對「店面預約工具」的啟示

1. **服務越到店、標準化、高頻、本地，SaaS 越穩；越到家、非標、低頻越容易死**
2. **走訂閱（SaaS），不走抽成**。河狸家殷鑑
3. **品牌信任靠實體店沉澱，不要試圖讓平台取代實體店**

## 四、台灣：碎片化市場、沒有龍頭

### 4.1 LineBooking.me

炫光螞蟻科技。官網業務電話仍是個人號，顯示為**中小型 SaaS，估計月營收 < 1,000 萬台幣等級，店家數千等級**。

### 4.2 StyleMap 美配

- 月活 **40 萬消費者**、設計師 **1 萬+**、週媒合 **5,000+ 次**
- 「美 Pay」金流服務抽 **10%**（對標 minimo 但 take rate 高一倍）
- 台灣最接近 HPB + minimo 混合體，但規模遠遠沒形成壟斷

### 4.3 客立樂 QLiEER、美業歐巴 BeautinQ、BookNow、夯客 Hotcake

近年台灣美業專用 SaaS：
- **客立樂（2013，iCHEF 策略投資）**：訂閱制
- **美業歐巴（威許移動）**：LINE 會員經營 + 預約
- **夯客 Hotcake、BookNow**：LIFF + LINE 預約

**全部高度依賴 LINE 生態**。

### 4.4 為何台灣沒有美業 SaaS 龍頭？

1. **市場總量不夠**：台灣美業 5-7 萬家，比日本 25 萬家少 4 倍，HPB 模式做不起來
2. **LINE 就是流量層**：小型 SaaS 只能當「後台」，沒人能兼做流量與後台
3. **設計師流動率高**：個人品牌 >> 店家品牌
4. **創投不青睞**

**這對新進者反而是機會**：一個做到 1 萬店家、有明確 LINE 生態差異化的 SaaS，就能成為事實標準。

## 五、東南亞：super app 失敗、獨立玩家掙扎

### 5.1 Gojek GoLife（2015-2020）

**2019/12 關閉 GoGlam 等；2020/7 全關**。原因：stagnant business performance + COVID-19 到家服務死亡。Gojek 砍 9% 員工，聚焦交通/外送/支付。

**super app 外掛美業失敗是全球通則**（Uber、Didi 都試過又放棄）。美業不是 super app 的高頻引流鉤子。

### 5.2 Vaniday（新加坡，Rocket Internet，2015 - 2019）

- 對標英國 Treatwell 跨六國
- 2019/12 全球 shutdown
- 原因：promotion-heavy → 毛利崩、Rocket Internet 模式東南亞水土不服、小店不用信用卡、習慣 LINE/WhatsApp 私訊

### 5.3 泰國 / Grab

- **Grab 從未真正做美業**（吸取 GoLife 教訓）
- **泰國本土 SaaS** 規模小，多數仍靠 **LINE Official Account 私訊**

## 六、重點分析題

### 6.1 為什麼日本 HPB「雜誌→平台」路徑只在日本成立？

三個結構條件缺一不可：
1. 日本「免費派發雜誌」文化
2. Recruit 的**廣告銷售 DNA**（6,000+ 業務代表掃街談刊登費）
3. 日本高密度城市 + 低流動小店生態

其他市場沒這三條件，只能走 take rate 或 subscription。

### 6.2 為什麼中國 O2O 美業到家失敗？

服務非標 + 手藝人跳單 + 低頻 + 安全顧慮。對你的啟示：**不要碰到家、不要碰服務流程本身。做好「店面預約 + CRM + 老闆手裡的 iPad/手機」就夠**。

### 6.3 韓國 super app 吃下預約後，獨立 SaaS 還有空間嗎？

**有，但只能做後台**：深度 CRM、POS、小店老闆手機 UX、多分店管理。

### 6.4 LINE 生態：護城河還是依賴風險？

**雙面刃**：
- **護城河**：台灣 95%+ 使用 LINE，LIFF + Rich Menu + Messaging API = 「零下載、零註冊、零密碼」完美入口
- **依賴風險**：LINE 政策可隨時改（2023 年官方帳號收費結構大改）
- **對沖策略**：把 LINE 當「最強渠道」而非「唯一渠道」；資料、核心 CRM、付款流沉澱在**自有資料庫**

### 6.5 台灣起家能打到日本／東南亞嗎？

**最佳路徑**：
```
台灣單店 → 台灣 1,000 家（驗證模型）
       → 泰國 100 家（同 LINE 生態、低競爭）
       → 日本個人工作室（minimo 錯位）
       → 東南亞 Zalo/WhatsApp 另闢架構
```

## 七、給你的專案的策略建議

| 維度 | 建議 |
|---|---|
| **Take rate vs 訂閱** | **走訂閱**（河狸家殷鑑，小店最怕「抽流水」） |
| **流量 vs 後台** | **先做後台（SaaS）**。流量靠老闆自己的 LINE 官方帳號 |
| **定價錨點** | 參考 SalonBoard（免費 + HPB 鎖定）、客立樂（NT$899-1,999/月）。合理訂閱 **NT$699-1,499/月** |
| **LINE 依賴** | 當最強渠道 + 差異化護城河，但資料自有 |
| **產品差異化** | **台灣美業 SaaS 普遍 UI 難用**。mobile-first PWA 是 HPB / 博卡做不到的——你已有此優勢 |
| **擴張順序** | 台灣 → 泰國 → 日本 minimo 錯位 → 回頭做大台灣 |
| **避免** | 到府服務 / 抽成模式 / 自建流量 marketplace / 多角化餐飲 |
