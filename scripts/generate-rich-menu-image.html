Usage Instructions

1. Choose a color theme above, then click "Save as PNG".

2. Run the setup script:

npx tsx scripts/setup-rich-menu.ts --image path/to/downloaded/rich-menu.png

3. Or place the image at scripts/rich-menu.png and run without arguments:

npx tsx scripts/setup-rich-menu.ts
